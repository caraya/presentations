Keyboard Navigation

  • → or ↓ to advance to the next slide or incrementally displayed item
  • ← or ↑ to go to the previous slide or incrementally displayed item
  • Ctrl→ or Ctrl↓ to jump to the next slide
  • Ctrl← or Ctrl↑ to jump to the previous slide
  • Home to go to the first slide, End to go to the last
  • CtrlG to jump to an arbitrary slide (by slide number or identifier)
  • Note that if a slide contains an <iframe>, when the content in the iframe is focused, Inspire.js shortcuts will not work. There is no way around that, you need to click on something from the actual slide to get focus out of the iframe before you use any Inspire.js keyboard shortcuts. Some plugins (e.g. live demo) add on-screen "Next" buttons for this reason.

Understanding JSON

What is JSON?

  • An international standard: ECMA 404: The JSON data interchange syntax
  • A way to exchange data between servers and clients
  • A way to configure WordPress block themes

Introducing theme.json

What is theme.json

  • Developed as part of the WordPress Gutenberg project
  • Provides configuration for block themes
  • Controls configuration of the full site editor

Example Code


    {
      "$schema": "https://schemas.wp.org/trunk/theme.json",         
      "version": 2,
      "settings": {},
      "styles": {},
      "customTemplates": [],
      "templateParts": [],
      "patterns": []
    }

Schema configuration attributes


      "$schema": "https://schemas.wp.org/trunk/theme.json",
      "version": 2
  • The $schema attribute tells the editor where to find the definition of the schema we'll use
    • This schema document will help editors like VS Code or WebStorm highlight errors as you type.
  • There are two different versions of the schema
    • We should always use the latest indicating version to be 2
    • The recommended value will change as the Gutenberg team releases new versions
